Responsive Slider

A simple script to create a responsive slider. Keyboard shortcuts, touch enabled, and expandable through lazy-loading methods. More documentation coming soon.

You can download or fork this at http://github.com/echodesigngrp/responsive-components

Demo 1

Features: pagination, keyboard access (right and left arrows) and touch

Demo 2

Features: infinite and custom pagination

Demo 3

Features: touch, automate, and no navigation

Demo 4

Features: keyboard access, touch, multiple slides per view, and custom next/previous text
