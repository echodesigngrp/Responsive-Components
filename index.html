Responsive Components

Slider

A simple script to create a responsive slider. Keyboard shortcuts, touch enabled, and expandable through lazy-loading methods. More documentation coming soon.

Image

This is a simple solution for mobile-first images. Specify the mobile image, and provide data- attributes for any higher res images needed. Includes support for retina images. This makes sure that the browser only loads what is necessary.

You can download or fork this at http://github.com/echodesigngrp/responsive-components
